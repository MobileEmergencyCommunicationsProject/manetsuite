OLSR GUI

OLSR GUI for monitoring and manipulating the NRL OLSR Daemon on your Meego/Harmattan phone. Use the menu to view routes, neighbors, olsr settings, and this help page.

Credits

OLSR GUI is part of the MANET Suite of the Mobile Emergency Communications Project. The MANET Suite is a collection of mobile phone applications for creating and using mobile ad hoc networks using smart phones. The Mobile Emergency Communications Project is a collection of mobile phone applications that help people to communicate under emergency conditions.
